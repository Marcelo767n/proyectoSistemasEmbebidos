# Presentación

En esta carpeta estarán albergados los códigos del proyecto. Aquí se guardará todo el código fuente, sketches, scripts y utilidades necesarias para el desarrollo, pruebas y despliegue del sistema embebido.

## Contenido esperado
- `arduino/` — Sketches y librerías para microcontroladores.
- `wifi/` — Firmware y configuraciones del módulo WiFi.
- `python/` — Scripts de procesamiento, visualización y pruebas.
- `utils/` — Herramientas auxiliares y scripts de soporte.

## Cómo empezar (rápido)
1. Abrir la subcarpeta correspondiente al componente que desea probar.
2. Para Arduino: compilar y subir con el IDE de Arduino o `arduino-cli`.
3. Para Python: crear un entorno virtual e instalar dependencias antes de ejecutar los scripts.
